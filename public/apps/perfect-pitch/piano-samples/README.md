# Piano Samples Directory

Place your piano sample WAV files in this directory.

## Required Files

The app will work with synthesized tones if files are missing, but for best audio quality, include:

- `C4.wav`
- `C#4.wav` or `Db4.wav`
- `D4.wav`
- `D#4.wav` or `Eb4.wav`
- `E4.wav`
- `F4.wav`
- `F#4.wav` or `Gb4.wav`
- `G4.wav`
- `G#4.wav` or `Ab4.wav`
- `A4.wav`
- `A#4.wav` or `Bb4.wav`
- `B4.wav`

## File Specifications

- Format: WAV
- Sample Rate: 44100 Hz (preferred) or 48000 Hz
- Octave: 4th octave (C4 = middle C)
- Duration: ~1-2 seconds recommended

## Getting Samples

You can:
1. Record your own piano samples
2. Download from free sample libraries
3. Generate using audio software
4. Use the app's built-in synthesized tones (no files needed)

The app will automatically fall back to synthesized tones if files are not found.
